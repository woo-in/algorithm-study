🧩 Problem 4 | 나는야 포켓몬 마스터 이다솜

🚀 접근법

처음 생각 : 배열에 넣자. 

N : 도감수록 포켓몬 갯수 , 1 <= N <= 100000 
M : 검색 포켓몬 갯수 1 <= M <= 100000 

배열에 넣는데 ,  O(N) 
포켓몬 번호로 이름 검색 , O(M) 

포켓몬 이름으로 번호를 검색 -> O(N*M) <--- 이 부분에서 시간초과가 나겠구나 

어떻게 줄일까 ? 
해쉬 맵 사용 (이름 - 번호)  
해쉬 맵에 넣는데 , O(N) 
해쉬 맵에서 검색하는데 , O(M)
